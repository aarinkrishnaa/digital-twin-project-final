# Smart Factory Machine Digital Twin

A full-scale Industry 4.0 Digital Twin system featuring:

- Real-time machine sensor simulation (RPM, vibration, temperature)
- MQTT live data streaming
- Digital Twin engine with:
    - Predictive maintenance (ML)
    - Anomaly detection
    - State estimation
- Professional dashboard built with Streamlit/Dash
- Optional 3D machine visualization (Three.js)
